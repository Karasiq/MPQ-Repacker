[B]MPQ RePacker[/B] - программа для пересборки карт Warcraft 3.

Возможности:
[LIST]
[*]Вставка чит-пака
[*]Оптимизация карты путём выкидывания из неё неиспользуемых файлов
[*]Замена некорректного war3map.w3i и создание пустого war3mapUnits.doo
[*]Возможность паковать карту X-Deprotect'ом
[*]Полная расширяемость с помощью плагинов
[*]Рандомная генерация активатора (дабл-клик на поле ввода активатора)
[*]Все настройки сохраняются (не надо каждый раз все настраивать заново)
[*]9 встроенных чит-паков; можно добавлять свои
[/LIST]

[B]SLKRecover[/B] - простой плагин для [B]MPQ RePacker[/B], использует консольную прогу [B]Silk Object Optimizer[/B] с опцией [U]EXPORT_OBJECT_DATA = 1[/U] для восстановления файлов [U]war3map.w3*[/U] из файлов [U]*.slk[/U]. Таким образом, можно (например) восстановить данные о юнитах и способностях в доте (проверено на 6.72c).
